<h2>DowntheSpot</h2>

<h3>Installation:</h3>

<h4>First of all:</h4>

- Install Node.js
- Install bower by typing: 'npm -g bower' in a console

<h4>Installing dependancies</h4>

- Open a console and type: 'npm install'
- Run the app: 'node app.js'
- Open a browser at this address: http://localhost:3000/
